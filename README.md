Motor PH Payroll System - Terminal Assessment Submission

COMPROG1 - A1101

Project Overview

The MotorPH Payroll System Phase 1 aims to develop a basic payroll application that displays employee details and automatically computes weekly salaries. The system focuses on simplifying payroll processing by calculating hours worked, gross wage, and net wage using standard deductions. This phase serves as the foundation for future enhancements in succeeding phases.

Effort Estimate and Project Timeline (Phase 1)

The primary effort in Phase 1 is concentrated on developing the payroll computation features of the Motor PH Payroll System. Moderate effort is required to implement the logic for calculating total hours worked, as this involves validating inputs and ensuring correct weekly totals. Greater effort is needed for the gross wage calculation, since it integrates hours worked with the assigned hourly rate and must produce accurate results.

The highest level of effort is allocated to the net wage computation, as this involves applying deductions correctly and ensuring that the final salary breakdown is precise and reliable. Significant effort is also dedicated to testing the payroll calculations to verify accuracy, consistency, and proper system behavior under different input scenarios.

Project Plan

The project will follow a structured weekly development schedule to ensure timely completion and proper implementation of system features. In Week 4, the focus will be on reviewing system requirements and creating wireframes to visualize the overall design and functionality of the application. This stage is essential in establishing a clear direction for development.

In Week 5, the system will be developed to present employee details, including employee number, name, and birthday. This will ensure that the system can properly display and manage basic employee information. By Week 6, the logic for calculating the total number of hours worked per week will be implemented, forming the basis for salary computation.

During Week 7, the system will compute the gross weekly wage based on the employee’s total hours worked and hourly rate. This will be followed in Week 8 by the implementation of net wage computation, where standard deductions will be applied to determine the final salary.

Weeks 9 to 10 will be dedicated to testing, debugging, and refining the system to ensure accuracy, reliability, and usability. Any errors or inconsistencies identified during testing will be resolved during this phase. Finally, in Week 11, the completed Phase 1 payroll system, along with all required documentation, will be finalized and submitted.


Objectives

The main objective of this project is to develop a functional payroll system using Java. It also aims to automate weekly salary computations and ensure accuracy in payroll calculations.

Expected Output

The expected output of this project includes a functional Phase 1 payroll system capable of accurately computing employee hours worked, gross wage, and net wage. Additionally, the project will produce clean and well organized source code along with complete documentation.

Scope and Limitations

The scope of the system includes employee information display, weekly salary computation, and basic deduction handling within a console based environment. However, the system does not include database integration, web or mobile interfaces, or advanced payroll features, as these may be considered for future development phases.
